Splinter: Python web application testing framework
a talk by Evgeny Shmarnev

All you probably know about Selenium (portable software testing framework for web applications), but what if I told you that you can test your web applications much faster and easier? I will show what Splinter is, why you should use it and demonstrate it's features.

Evgeny Shmarnev

Evalle eshmarnev

Evgeny currently works for SUSE as OpenStack Automation Engineer. He is in charge of running the CI and build operations and most of the related infrastructure for the SUSE OpenStack Cloud project. He is an active open-source contributor (OpenSUSE, Docker and various python-related projects).
